    - 타이틀 작성해야 한다.
1. 기본 디테일
    - 두개의 버튼이 있음
        - 디테일 페이지에 들어가면 버튼에 이전 셀 text값이 들어간다.
            - 해당 버튼을 눌렀을 때 이전 테이블 뷰로 들어간다. 
        - 나머지 하나는 모달을 띄워준다. 
            - 모달이 완전히 띄워졌을 때 텍스트가 바뀐다.
1. 메인에는 두개의 텝이 있다.
    - 탭을 누를때 기본 메인 페이지로 가야 한다.
    - 두번째 버전 디테일은 네비게이션 바를 없앤다.
        - 아웃라인 자체를 없앰
